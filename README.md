# ⚡ NexaCalc – Modern Calculator Web App

A modern, responsive, and multi-functional calculator web application built using HTML, CSS, and JavaScript.  
Designed with a clean UI and smooth user experience.

## 🚀 Live Demo
🔗[ https://your-username.github.io/nexacalc/](https://thakurshab-dev.github.io/nexacalc/)

---

## 📌 Features

### 🧮 Basic Calculator
- Addition, Subtraction, Multiplication, Division
- Decimal support
- Clear (C) & Delete (⌫)

### 🔢 Binary Calculator
- Binary Addition, Subtraction, Multiplication
- Decimal ↔ Binary Conversion
- Input validation (only 0 & 1)

### 📊 Advanced Functions
- Square (x²)
- Cube (x³)
- Square Root (√)
- Percentage (%)

### 🧾 Extra Features
- Calculation History
- Responsive Design (Mobile + Desktop)
- Smooth UI interactions
